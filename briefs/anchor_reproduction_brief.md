# Anchor reproduction — trust check for the Stream A pooling pipeline

**Purpose.** Before extending the designed-study synthesis, verify that this project's
meta-analysis pipeline recovers a *published* GLP-1-RA / suicidality pooled estimate from
its own per-study data. If the pipeline reproduces the anchor within its confidence
interval, the extended synthesis is credible; any divergence is flagged here.

## Anchor selected

**Chen et al., *Impact of GLP-1 Receptor Agonists on Suicide Behavior: A Meta-Analysis
Based on Randomized Controlled Trials*, J Diabetes 2025;17(9):e70151.**
DOI [10.1111/1753-0407.70151](https://doi.org/10.1111/1753-0407.70151) · PMID 40887719 · PMC12399406 · CC BY.

Chosen because it (a) is GLP-1 RA × suicidal behaviour specifically, (b) reports a pooled
RR with CI and full heterogeneity statistics, and (c) publishes a complete per-study forest
plot (25 RCTs) that allows the 2×2 table to be rebuilt independently rather than re-quoted.

*Note on the JAMA Psychiatry 2025 alternative.* The higher-profile Ebrahimi/Batlle et al.
JAMA Psychiatry meta-analysis (27/144 RCTs; incidence-rate ratio 0.76 [0.48–1.21]) was
considered as the primary anchor, but its per-study event counts sit in a subscription
supplement, and it carries a July-2025 person-time correction plus a published "errors in
numbers" exchange. It remains a valuable second anchor for the person-time (rate-ratio)
pipeline; the J Diabetes RCT set is used here because it is fully reproducible from open data.

## Method

- Per-study events and arm sizes for all 25 trials were transcribed from the Figure 2 forest
  plot (retrieved as the open-access figure image via Europe PMC / EMBL-EBI) into
  `anchor_jdiabetes_2x2.csv`. All column totals reconcile exactly with the published figure
  (Σ exposure events = 35, Σ control events = 36; Σ N = 42 172 vs 39 223).
- Pooled with a Mantel–Haenszel random-effects model on the risk ratio, DerSimonian–Laird
  τ², 0.5 continuity correction for single-zero studies, double-zero studies excluded
  (RevMan/`meta::metabin` defaults) — matching the anchor's stated methods.
- 23 of 25 studies estimable (Ishii 2020 and Tamborlane 2022 are double-zero → dropped;
  df = 22).

## Result — exact reproduction

| Statistic | Published | Reproduced | Match |
|---|---|---|---|
| Pooled RR (random) | 0.84 | 0.844 | ✓ |
| 95% CI | 0.54–1.32 | 0.541–1.316 | ✓ |
| Heterogeneity Q (χ²) | 11.24 | 11.24 | ✓ |
| df | 22 | 22 | ✓ |
| I² | 0% | 0.0% | ✓ |
| Z (overall effect) | 0.75 | 0.75 | ✓ |
| P (overall) | 0.46 | 0.45 | ✓ |
| Total events | 35 vs 36 | 35 vs 36 | ✓ |

**Verdict: PASS.** The pipeline recovers the anchor to the reported precision on every
statistic — point estimate, interval, heterogeneity, and event totals all coincide. No
divergence to flag. The Stream A pooling machinery is validated for the extended synthesis.

## Interpretation (for the triangulation, not a Stream-A conclusion yet)

Within RCTs, GLP-1 RA exposure shows **no association** with suicidal behaviour (RR 0.84,
CI crosses 1), with **zero heterogeneity** — the trials agree. This is the designed-evidence
baseline against which the FAERS spontaneous-report signal (Stream B) will be compared. The
event counts are very low in both arms, so the RCT evidence is precise about the *absence of
a large effect* but underpowered for rare-event subgroups — a caveat that will matter when
reconciling against the disproportionality signal.
